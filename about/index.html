    About me

    Hi and welcome to my blog silvermarkg labs.

    My name is Mark Goodman and I live in the UK. I work as an Infrastructure Systems Engineer. My main technical focus is on deployment and management of end user computing services using Microsoft technologies such as Configuration Manager, Intune, Microsoft 365 and Azure. I also like to automate these technologies using PowerShell, Microsoft Graph, Logic apps and Azure functions.
